# 🏦 Sistema de Controle Bancário em Java

Aplicação desenvolvida em **Java** para simular um sistema bancário simples, permitindo o gerenciamento de contas e operações financeiras básicas via terminal.

---

## 🚀 Funcionalidades

- Criação de contas bancárias  
- Depósito de valores  
- Saque de valores  
- Consulta de saldo  
- Interação via terminal  

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)  
- Classes e objetos  
- Encapsulamento  
- Estruturas condicionais (`if/else`)  
- Laços de repetição  
- Entrada e saída de dados  

---

## 🛠️ Tecnologias

- Java
